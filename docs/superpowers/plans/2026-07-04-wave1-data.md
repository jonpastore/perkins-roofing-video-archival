# Wave 1 — Data Completeness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Ingest the entire 832-video channel (all long-form + 648 Shorts), transcribe via free local Whisper (VAD-skipping near-silent Shorts), and embed everything with `gemini-embedding-001` (3072-dim) into Cloud SQL pgvector.

**Architecture:** All code buildable now against the Foundation adapters; the **full live run needs prereq P1 (GCP billing)**. STT runs free on cerberus (faster-whisper); only embeddings hit Vertex (≈ a few dollars for 832 videos). Everything is a resumable Cloud Run Job keyed by `ingestion_runs`.

**Tech Stack:** yt-dlp, faster-whisper (cerberus RTX 5090), WebRTC/Silero VAD, Vertex `gemini-embedding-001`, Cloud SQL pgvector HNSW, Cloud Run Jobs.

## Global Constraints

- Inherits all Wave 0 constraints (Vertex-only, 3072-dim, coverage gate, idempotency).
- **STT is free/local by default** (Whisper on cerberus); GCP STT v2 is fallback only.
- **VAD-skip near-silent Shorts** to avoid transcribing music-only/silent clips — bound cost/noise.
- Re-embed **all 832** with the single prod model; discard the 161 dev nomic vectors.
- Licensed-music Shorts: transcript may be empty/muted — record `source` + a `skipped_reason`.
